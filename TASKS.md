## 1 Install conda 

## 2 Create Python Env. using conda
``` conda create --name myenv python=3.8 ```

## 3 Activate Python Env.
``` conda activate myenv ```

## 4 Install requirements in the above created env.
``` pip install -r requirements.txt ```
